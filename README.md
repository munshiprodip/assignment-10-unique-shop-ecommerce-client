## Unique Shop BD (Full Stack Project)

**_USING_**

**Client Side**

- React
- React Router Dom
- Bootstrap
- Fontawesome
- Firebase

**Server Side**

- MongoDB
- Mongoose
- Express js
- Heroku

**_FEATURES_**

- Login with Google
- Signup with Email and Password
- Login with created credential
- Protected Rout
- Redirect after Authentication
- Add product
- Edit product
- Delete product
- Checkout & view order list
